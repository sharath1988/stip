Error

The application encountered an error:
If you haven't been actively using the application, it could be you were timed out for security reasons and need to restart the application.
If the network is experiencing heavy volume, it could be the application timed out waiting for page data to load.
In these cases, close all open web browser windows, wait a few minutes, then try again.

If, after closing your web browser (i.e. Firefox or Internet Explorer), you can repeat the error then send an email message to the Business Application Development Group.
Email: hs-support@ucsd.edu
